-------------------------------------------------------------
Basic Java console app able to store items records in a file.
-------------------------------------------------------------

PROJECT TITLE: LocalInventoryManagerApp (LIM ver 1.0)
PURPOSE OF PROJECT: Java skills testing
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: Run Main.class
AUTHORS: George Giurgiu
USER INSTRUCTIONS:
  Console commands:
  add item_name doc_type_date nr_units (adds an item by name (description - one word), document type and/or date, and nr. of units)
  list (displays the list of items by index)
  remove index_nr nr_units (removes nr. of units from the item at that index)
  update (updates into the app the list previously saved in E:\items)
  exit (exits the app)
